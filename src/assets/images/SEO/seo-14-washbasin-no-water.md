## Bai viet nhap (tieng Nhat)

# 洗面所の水が出ない原因は？凍結・故障・元栓のチェックポイント

朝、顔を洗おうとして蛇口をひねっても水が出ない。お湯側は普通に出るのに、水だけ止まっている。あるいはその逆で、お湯だけ出ない。突然のことだと、何から確認すればいいか迷いますよね。

洗面所の水が出ない原因は、季節や使用状況によってパターンが変わります。冬場は凍結、引っ越し直後なら止水栓、長く使った家なら水栓本体の劣化や配管の詰まりなど、状況に合わせた切り分けが大切です。原因を絞り込めば、自分で復旧できるのか、業者を呼ぶべきかを判断しやすくなります。

この記事では、洗面所の水が出ないときに考えられる原因と、自分でできるチェックの手順、業者へ相談したほうがよいケースまでをわかりやすく解説します。

## 1. 水が出ないときに最初に確認したいこと

修理を呼ぶ前に、状況を整理しておくと原因を絞りやすくなります。

・水・お湯のどちらが出ないのか、両方とも出ないのか
・洗面所だけか、キッチン・お風呂・トイレも出ないか
・前日や前回までは普通に使えていたか
・季節や前日の気温は（特に冬場の凍結リスク）
・引っ越し直後など、止水栓の状態が分からない状況か

家全体で水が出ない場合は、元栓・断水・水道管側の問題が疑われます。洗面所だけで、しかも水かお湯のどちらか片方だけ出ないなら、洗面台側に原因がある可能性が高くなります。

## 2. 考えられる主な原因

洗面所の水が出ない原因は、外部要因・止水栓・水栓本体・配管の4つに大きく分けられます。

### 2.1. 外部要因（断水・凍結）

家の外側で起きている問題で、利用者側で復旧できないものもあります。

・地域の断水・水道工事
・寒波による配管の凍結（特に夜間〜早朝）
・元栓（屋外メーター付近）が閉まっている
・大雨や災害後の供給制限

### 2.2. 止水栓の状態

洗面台の下にある止水栓が閉まっていたり、開きが浅かったりすると、水が十分に流れません。

・引っ越し・清掃後に閉めたままになっている
・修理時に閉めて、再開し忘れている
・止水栓本体が経年で固着している
・お湯側だけ閉まっていて湯が出ない

### 2.3. 水栓本体の故障・詰まり

蛇口そのものの問題は、見た目では分かりにくいことがあります。

・吐水口の網（ストレーナー）に水垢・ゴミが詰まっている
・シングルレバー水栓の内部カートリッジの劣化
・温度調節機能（サーモスタット）の不具合
・本体の寿命（10年前後が目安）

### 2.4. 給湯器・配管側の問題

お湯側だけ出ない場合は、給湯器・お湯配管側の可能性があります。

・給湯器の電源が入っていない、エラーが出ている
・ガス・電気が止まっている
・お湯配管の凍結
・給湯器そのものの故障

## 3. 自分でできる確認の手順

利用者側で安全にできる範囲を順番に進めます。状況に合わせて該当する手順から確認しましょう。

### 3.1. 他の蛇口の状態を確認する

キッチン、お風呂、トイレなど他の場所で水が出るかを確認します。家中で出ないなら断水・元栓・凍結が、洗面所だけなら洗面台側が原因と切り分けられます。

### 3.2. 凍結を疑う（冬場）

外気温が氷点下近くまで下がった夜の翌朝に出ないなら、配管の凍結の可能性が高くなります。

・気温が上がるのを待ち、自然解凍させるのが最も安全
・急いでいる場合は、配管にタオルを巻き、ぬるま湯（40〜50℃）をゆっくりかける
・熱湯を直接かけるのは配管の破裂につながるため避ける
・水抜き栓がある住宅では、操作を間違えないよう取扱説明書を確認する

### 3.3. 止水栓・元栓を確認する

洗面台の下を開け、止水栓（給水・給湯の2つがあることが多い）が開いているかを確認します。マイナスドライバーで反時計回りに回すと開きます。完全に閉まっていた場合は、ゆっくり開けて水量を確認します。

家全体で出ないなら、屋外メーターの元栓も確認します。

### 3.4. 吐水口の網を掃除する

水栓本体の問題かを切り分けたい時は、吐水口の先端を回して外し、内側の網（ストレーナー）に水垢やゴミが詰まっていないかを見ます。歯ブラシでこすり、流水で洗ってから元に戻します。

### 3.5. 給湯器を確認する(お湯だけ出ない場合)

お湯だけ出ない場合は、給湯器のリモコンに電源が入っているか、エラーコードが出ていないかを確認します。リモコンの電源が落ちている時は、入れ直してから蛇口を試します。

エラーコードが出ている、リモコンに反応がないといった場合は、給湯器本体側の問題が疑われるため、別記事「給湯器が動かない原因と確認したい5つのポイント」も参考にしてください。

ここまでの確認で改善しない場合や、前述のように凍結のあとに水を出したら漏れている・止水栓を開けても水量が戻らないといった状況は、利用者側で対応できる範囲を超えています。配管の破損や水栓本体の故障が疑われるため、業者に相談したほうが安全です。

## 4. こんなときは早めの相談を

次のような状況に当てはまる場合は、自分で何度も試さず、早めに業者へ相談することをおすすめします。

・止水栓・元栓を確認しても水が出ない
・凍結のあと水を出したらシンク下や床が濡れている
・水栓のレバー・ハンドルが固くて動かない
・水栓本体や床下から水音・水漏れの気配がある
・お湯側だけ給湯器のエラーで動かない
・設置から10年以上経った水栓・給湯器
・断水復旧後、洗面所だけ水が戻らない

特に凍結のあとに濡れが出ているケースは、配管にひびが入っている可能性があります。一度水を止めて元栓を閉め、無理に通水を続けないでください。水道メーターのパイロット（小さな羽根）が回り続けている場合も、見えない場所での漏水サインです。

| 自分で対応しやすいケース | 業者へ相談したいケース |
| --- | --- |
| 止水栓・元栓が閉まっていて開けて復旧 | 元栓を開けても水が出ない |
| 凍結を自然解凍・ぬるま湯で復旧 | 凍結後に床・シンク下が濡れている |
| 吐水口の網のゴミ除去で水量が戻る | 水栓レバーが固くて動かない、漏れている |
| 給湯器の電源・エラー確認で再開 | 給湯器エラーが頻発する、寿命に近い |
| | 設置から10年以上、複数箇所で不調 |

洗面所は朝の身支度や手洗いなど、毎日使う場所です。止水栓や凍結の確認をしても改善しない、原因がわからず不安というときは、無理に通水を繰り返さず一度ご相談ください。

水栓の修理・交換から配管の点検、凍結被害の調査・給湯器の対応まで、年中無休・出張対応でサポートしています。
