# お風呂で水漏れが発生！場所別の原因と修理費用の目安

お風呂掃除中にシャワーホースの根元から水がにじんでいるのに気づいた。浴室の床がいつもより乾きにくく、目地の隅が常に湿っている。階下の方から「天井に染みが出ている」と連絡が来た。お風呂の水漏れは、見えにくい場所で進行しやすく、気づいたときには被害が広がっているケースもあります。

お風呂の水漏れは、起こっている場所で原因も対処も大きく変わります。シャワー・蛇口まわりは比較的見つけやすい一方、浴槽の下や床下の配管は目視できず、専門の調査が必要になることもあります。場所と症状から原因を絞り込めば、応急処置で済むのか、修理が必要なのかを判断しやすくなります。

この記事では、お風呂の水漏れが起こる場所別の原因と、自分でできる確認、修理費用の目安、業者へ相談したほうがよいケースまでをわかりやすく解説します。

## 1. 水漏れに気づいたときに最初に確認したいこと

漏れに気づいたら、被害を抑えるために次の点を整理します。

・どこから水が漏れているか（シャワー／蛇口／浴槽／床／壁際）
・水を使った時だけ漏れるか、使っていない時も濡れているか
・浴室の外側（脱衣所・廊下・階下）に染みが広がっていないか
・カビ臭・木材の腐ったようなにおいがしないか
・止水栓・元栓の場所を把握しているか

水を使っていないのに濡れている場合は、給水側の配管に問題がある可能性が高くなります。逆に水を流した時だけ濡れる場合は、排水側です。脱衣所・階下に染みが広がっているケースは、見えない場所での漏水が進んでいるサインなので、利用者側の対応では追いつきません。

## 2. 場所別によくある原因

### 2.1. シャワー・水栓まわり

毎日使う部分で、経年劣化が現れやすい箇所です。

・シャワーホースの根元・ジョイント部のパッキン劣化
・シャワーヘッドの取り付け部のゆるみ
・水栓のレバー・ハンドル内部のカートリッジ劣化
・サーモスタット部のシール劣化

### 2.2. 浴槽まわり

浴槽そのものや、浴槽と床の取り合い部分から漏れるケースがあります。

・浴槽のエプロン内部（外せるカバーの中）に水が回っている
・浴槽と床の境目のコーキング（シリコン）の劣化・剥がれ
・浴槽の排水栓（ポップアップ式）のパッキン不良
・浴槽の追い炊き配管の接続部からの漏れ

### 2.3. 床・壁・目地

ユニットバスでは目に見える接続部・コーキング、在来工法のお風呂ではタイル目地が劣化ポイントになります。

・床と壁の継ぎ目のコーキング切れ
・タイル目地のひび割れ
・ドア下部のパッキン劣化
・床下点検口から床下に水が回っている

### 2.4. 給湯器・追い炊き配管

外側の給湯器・配管から漏れているケースは、お風呂内部ではなく屋外で発見されることが多くなります。

・給湯器本体の接続部のパッキン劣化
・追い炊き配管の劣化・接続不良
・凍結による配管のひび割れ

## 3. 自分でできる確認と応急処置

利用者側で安全にできる範囲を整理します。

### 3.1. 止水栓または元栓を閉める

被害を広げないために最優先で行うのは、水を止めることです。浴室の水栓に止水栓があるタイプなら、それを閉めます。場所がわからない、止まらない場合は、家全体の元栓を閉めます。

### 3.2. 漏れ場所を観察する

水気を一度しっかり拭き取り、新しいタオルやペーパーを敷いて、どこから新しく水が出てくるかを目視で観察します。「水を使っている時だけ」「使っていなくても」のどちらに当てはまるかで、給水側か排水側かを切り分けます。

### 3.3. シャワーホース・ヘッドの取り付けを確認する

シャワーヘッドの根元やホースの両端のナットが手で回せるほどゆるんでいる場合は、軽く締め直すと止まることがあります。締めすぎはパッキンを潰してしまうため、力を入れすぎないように気をつけます。

### 3.4. コーキング・目地の状態を見る

浴槽と床の継ぎ目、タイル目地にひびや剥がれがないか目視で確認します。表面的な隙間は補修剤で一時的にふさげることもありますが、見えない奥側にすでに水が回っている場合は根本解決にならないため、応急処置にとどめます。

### 3.5. 浴槽エプロンの中は無理に開けない

浴槽の外側カバー（エプロン）は外せるタイプもありますが、内部の配管・断熱材を傷めるリスクがあります。前述のように見えない場所の漏水は専門調査が必要なため、自己判断で開けたり手を入れたりせず、業者に診てもらうのが安全です。

ここまでの応急処置で改善しない場合や、止水栓を閉めても水が出続ける・階下や脱衣所に染みが広がっているといった状況は、利用者側で対応できる範囲を超えています。元栓を閉めた状態で業者に連絡してください。

## 4. 修理費用の目安と費用が変わる要素

実際の費用は機種・症状・部品の状態で変動しますが、大まかな目安は次のとおりです（あくまで参考値です。実際は現地確認と見積もりが必要です）。

・シャワーホース・ヘッドの交換　➞ 数千円〜2万円程度
・水栓のカートリッジ・パッキン交換　➞ 1〜3万円程度
・水栓本体の交換　➞ 3〜6万円程度
・浴槽と床の境目のコーキング打ち直し　➞ 1〜3万円程度
・浴槽の排水栓・パッキン交換　➞ 1〜3万円程度
・追い炊き配管の修理　➞ 数万円〜
・床下・壁内の配管修理　➞ 規模により大きく変動

費用が大きく変わる要素は、次のような点です。

・設置から10年以上経っているか（複数箇所の同時劣化）
・ユニットバスか在来工法か（工事の難易度が違う）
・浴槽エプロンの中・床下まで被害が及んでいるか
・階下への漏水被害があるか
・夜間・休日対応か

特に階下漏水が起きているケースは、自宅の修理費用とは別に、階下の修繕費用負担が発生する可能性があります。火災保険の「水濡れ」補償の対象になるケースもあるため、加入している保険の内容も確認しておくと安心です。

## 5. こんなときは早めの相談を

次のような状況に当てはまる場合は、自分で粘らず早めに業者へ相談することをおすすめします。

・止水栓・元栓を閉めても漏れが止まらない
・浴槽の下や床下から水音・湿気を感じる
・脱衣所・廊下に水のしみが広がっている
・階下の天井に染み・水滴が出ている
・浴室全体がカビ臭い、木材の腐ったにおいがする
・水道使用量が急に増えている
・設置から10年以上経過し、複数の不調が出ている
・凍結後に漏れ始めた、給湯器周りからも漏れている

特に階下漏水・床下漏水・カビ臭が広がっているケースは、見えない場所で進行している可能性があるため、早期の点検が結果的に修理費用を抑えることにもつながります。

| 自分で対応しやすいケース | 業者へ相談したいケース |
| --- | --- |
| シャワーヘッド・ホースの取り付けゆるみ | 止水栓を閉めても漏れが止まらない |
| 表面的なコーキングの小さなひび | 浴槽下・床下から水音・湿気がする |
| 一時的な水滴で止水栓を閉めると止まる | 脱衣所・階下に染みが広がっている |
| シャワーホースの交換で改善 | カビ臭・腐臭が浴室全体に広がる |
| | 設置から10年以上で複数箇所が漏れている |

お風呂の水漏れは、表面に出ないまま床下や階下に被害が広がるケースが多く、気づいた段階で早めに動くほど修理費用も小さく済む傾向があります。応急処置で止水しても改善しない、原因がわからず不安というときは、無理に部品交換を試さず一度ご相談ください。

水栓・シャワーの修理交換から、浴槽・コーキング・配管の調査、階下漏水時の調査対応まで、年中無休・出張対応でサポートしています。
