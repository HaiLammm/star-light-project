# ブレーカーが上がらない時の原因と安全に復旧する手順を解説

夜中に急に家中が真っ暗になり、分電盤を見に行ったらブレーカーが下がっていた。いつものように上げ直そうとしても、レバーがすぐに戻ってしまう。または、上がったように見えてもまた落ちる。そんなとき、無理に何度も操作するのは危険です。

ブレーカーが上がらない原因は、漏電・配線の異常・本体の劣化など、放置できないものが多くあります。安全に復旧するには、どのブレーカーが上がらないのかを見極め、原因を切り分けてから手順を踏むことが大切です。

この記事では、ブレーカーが上がらないときに考えられる原因と、自分で安全に試せる復旧手順、業者へ相談したほうがよいケースまでをわかりやすく解説します。

## 1. 上がらないときに最初に確認したいこと

操作を始める前に、状況を整理しておくと原因を絞りやすくなります。

・どのブレーカーが下がっているか（アンペア／漏電／安全のどれか）
・「漏電」表示やトリップマークが出ていないか
・家全体が停電しているか、特定の部屋だけか
・落ちる直前に使っていた家電はあるか
・水濡れや雷、強風など外的要因がなかったか

特に漏電ブレーカーが落ちていて、レバーを上げてもすぐ戻る場合は、配線や家電のどこかで漏電が起きているサインです。原因を放置したまま無理に復旧させると、感電や火災の危険があります。

## 2. ブレーカーが上がらない主な原因

ブレーカーが戻らない理由は、漏電・回路異常・本体劣化の3パターンに大別できます。

### 2.1. 漏電が発生している

最も多いのが、家のどこかで漏電が起きているケースです。

・洗濯機・電子レンジなど水回り家電の絶縁劣化
・コンセント内部に水・湿気が入った
・配線の被覆が傷ついて電気が漏れている
・古い家電本体内部のショート
・落雷で配線や家電がダメージを受けた

### 2.2. 回路に過剰な負荷が残っている

落ちた直後に大容量家電のスイッチが入ったままだと、上げてもすぐにまた落ちてしまいます。

・電子レンジ・ドライヤー・IH調理器などが運転中の状態のまま
・故障した家電がショートに近い状態でつながっている
・タコ足配線のまま容量を超えている

### 2.3. ブレーカー本体・分電盤の劣化

設置から長期間経過した分電盤では、ブレーカー本体の不具合も起こります。

・接点が摩耗して接触不良を起こしている
・経年劣化でレバーが固く、正常に動かない
・分電盤内部に錆や湿気がある
・部品の寿命（15〜20年が目安）

## 3. 安全に復旧する手順

漏電や負荷の問題を切り分けるための基本手順を、順を追って説明します。利用者側で安全にできる範囲のみを扱っています。

### 3.1. すべての家電のスイッチを切る

復旧操作を始める前に、家中の家電のスイッチを切り、コンセントから抜くのが基本です。特に、エアコン・冷蔵庫・洗濯機・電子レンジなどの大型家電や、水回りの家電を優先します。これで「家電側の負荷や異常」を一度ゼロにできます。

### 3.2. 安全ブレーカーをすべて下げる

分電盤の右側に並んでいる安全ブレーカー（部屋・回路ごとのブレーカー）を、すべて下げた状態にします。これにより、どの回路に問題があるかを後で切り分けやすくなります。

### 3.3. 主幹・漏電ブレーカーを上げる

家全体の電気を一度止めた状態で、左側のアンペアブレーカー → 漏電ブレーカーの順で上げます。漏電ブレーカーが正常に上がるかをここで確認します。

漏電ブレーカーがすぐに戻ってしまう場合は、ブレーカー本体の不具合や、分電盤内部の漏電が疑われます。自分での復旧はここで止め、業者に相談してください。

### 3.4. 安全ブレーカーを1つずつ上げる

漏電ブレーカーが正常な状態で、安全ブレーカーを1つずつ順番に上げていきます。途中で漏電ブレーカーが落ちた回路があれば、その回路に漏電源があると分かります。

該当回路だけ下げたままにし、他の回路を復旧させたら、漏電源となっている回路の家電をひとつずつコンセントから抜いた状態で再度試し、原因の家電を特定します。原因が分かったら、その家電は使用を止め、業者または家電メーカーに相談します。

ここまでの確認で漏電源が特定できない、または何度試しても漏電ブレーカーが上がらない場合は、配線側や分電盤本体の異常が疑われます。前述のように利用者側で対応できる範囲を超えているため、無理に通電を繰り返さず業者に依頼するのが安全です。

## 4. こんなときは早めの相談を

次のような状況に当てはまる場合は、自分で何度も復旧を試さず、早めに業者へ相談することをおすすめします。

・家電をすべて外しても漏電ブレーカーが上がらない
・特定の回路だけ漏電ブレーカーが繰り返し作動する
・分電盤やコンセントから焦げ臭いにおいがする
・分電盤を触ると熱を感じる、異音がする
・ブレーカーのレバー自体が固くて動かない、戻らない
・水濡れ・落雷・強風のあとから不調が続いている
・設置から15年以上経過した分電盤

特に、焦げ臭さ・煙・分電盤の発熱を伴うケースは、火災のリスクが極めて高い状態です。すぐに主幹ブレーカーを切ったまま触らず、業者を呼んでください。分電盤の内部や配線の修理・交換は、電気工事士の資格が必要な作業のため、自分で開ける・触るのは避けたほうが安全です。

| 自分で対応しやすいケース | 業者へ相談したいケース |
| --- | --- |
| 大容量家電を切ってから順に復旧して直る | 家電を外しても漏電ブレーカーが上がらない |
| 原因の家電を特定でき、使用を止められる | 焦げ臭さ・煙・分電盤の発熱がある |
| タコ足配線を整理して再発しない | レバーが固い・戻らないなど分電盤本体の不調 |
| 一時的な負荷集中で落ちただけ | 水濡れ・落雷の後から復旧できない |
| | 設置から15年以上の古い分電盤 |

ブレーカーが上がらない状態が続くと、生活への影響が大きいだけでなく、漏電を放置することで火災や感電のリスクが高まります。家電を外しても直らない、原因がわからず不安というときは、無理に通電を繰り返さず一度ご相談ください。

漏電調査から分電盤の交換、配線の見直しまで、年中無休・出張対応でサポートしています。
