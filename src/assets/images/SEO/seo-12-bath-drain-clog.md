# お風呂の排水溝が流れない！詰まりの原因と自分でできる解消法

シャワーを使っているうちにお湯が足元に溜まってきた。湯船を抜いたら、いつもよりかなり時間がかかって流れていく。排水口から「ゴポゴポ」と空気の音が聞こえる。お風呂の排水まわりは、毎日使う場所だけに、少しの違和感もすぐに気になりますよね。

お風呂の排水溝が流れない原因の多くは、髪の毛と石鹸カス、皮脂汚れが絡み合った詰まりです。表面のヘアキャッチャーだけでなく、その奥のトラップや配管にも汚れが溜まることで、流れが悪くなっていきます。軽度の詰まりは利用者側で解消できることが多いので、まずは正しい順序で確認していきましょう。

この記事では、お風呂の排水溝が流れない原因と、自分でできる解消法、業者へ相談したほうがよいケースまでをわかりやすく解説します。

## 1. 流れないと感じたときに最初に確認したいこと

排水が悪くなったと感じたら、まず現状を整理しておきます。

・流れが悪いのは浴槽の排水か、洗い場の排水か、両方か
・水がまったく流れないのか、ゆっくり流れるのか
・排水口から悪臭やゴポゴポ音がしていないか
・最後にヘアキャッチャーや排水口を掃除したのはいつか
・洗面所・キッチンの排水も同時に悪くなっていないか

特に、家の中の複数箇所の排水が同時に悪くなっている場合は、お風呂単体ではなく宅内全体の排水管・公共桝側の問題が疑われます。この場合は利用者側の掃除では解決しないため、業者への相談が現実的です。

## 2. 詰まりの主な原因

お風呂の排水溝が流れなくなる原因は、汚れの種類と詰まる場所で整理できます。

### 2.1. ヘアキャッチャー（目皿）の汚れ

排水口の一番上にあるパーツで、髪の毛をキャッチする役割を持っています。

・髪の毛が絡まってドーム状に固まっている
・石鹸カスや皮脂が髪の毛に絡みついて固まっている
・シャンプー・トリートメントの容器のシールやキャップが落ちている

### 2.2. 排水トラップ内部の汚れ

ヘアキャッチャーの下にある、封水を貯めて下水のにおいを防ぐ部分です。ここに汚れが溜まると、目に見えない場所で流れが落ちます。

・ぬめり（バイオフィルム）が内壁に厚く付着
・石鹸カス・皮脂が固まったヘドロ状の汚れ
・カビによる詰まり
・小物（ヘアピン・指輪・ピアスなど）が落ちて引っかかっている

### 2.3. 排水管・配管側の詰まり

トラップを超えて配管側で詰まっているケースは、利用者側で対処しにくくなります。

・長年蓄積した皮脂・石鹸カスの硬化
・配管接続部のずれや勾配不良
・配管内に異物が落ちて引っかかった
・浄化槽・公共桝の側で流れが悪くなっている

### 2.4. 浴槽側の排水栓・パッキンの劣化

浴槽の排水が悪いケースでは、ヘアキャッチャー以外にも原因があります。

・ポップアップ式排水栓のパッキン劣化や噛み込み不良
・浴槽の排水ホースの変形・劣化

## 3. 自分でできる解消の手順

利用者側で安全にできる範囲を順番にまとめます。多くの場合、軽度の詰まりはここまでで解消します。

### 3.1. ヘアキャッチャーを取り外して掃除する

排水口のフタを外し、ヘアキャッチャーを取り出します。絡みついた髪の毛と汚れを古い歯ブラシやスポンジで取り除き、流水で洗い流します。汚れが固まっている場合は、お風呂用中性洗剤に少しつけ置きしてからこすると落ちやすくなります。

### 3.2. トラップ部分を分解して洗う

機種によりますが、ヘアキャッチャーの下にあるトラップ部分（封水筒）は手で回して外せるものが多くあります。外したパーツのぬめりを中性洗剤で洗い、内部の溝に詰まった汚れも歯ブラシで落とします。元に戻すときは、向き・差し込みの順番を間違えないようにします。

### 3.3. 重曹とクエン酸でぬめりを分解する

機械的な掃除で取りきれないぬめりには、重曹とクエン酸（または酢）の組み合わせが有効です。

・排水口に重曹を1カップほど振りかける
・上からクエン酸を溶かしたぬるま湯を注ぐ
・泡立ったら30分ほど置く
・最後に40〜50℃のお湯でしっかり流す

熱湯（60℃以上）は配管を傷める可能性があるため、必ずぬるめのお湯を使います。

### 3.4. 市販のパイプクリーナーを正しく使う

汚れがしつこい場合は、市販の液体パイプクリーナーを使います。使用量と放置時間は商品の表示通りに守り、長時間放置しすぎないことが重要です。複数の薬剤を混ぜると有毒ガスが発生するため、塩素系と酸性のものを併用しないでください。

ここまでの手順で改善しない場合や、前述のように家の中の複数箇所で同時に排水が悪い・配管側で詰まっている可能性が高い場合は、利用者側で対応できる範囲を超えています。市販品の連用は配管を傷める恐れもあるため、無理に続けず業者に相談したほうが安全です。

## 4. こんなときは早めの相談を

次のような状況に当てはまる場合は、自分で粘らず早めに業者へ相談することをおすすめします。

・ヘアキャッチャー・トラップを掃除しても流れが改善しない
・市販のパイプクリーナーを試しても変わらない
・お風呂だけでなく洗面所・キッチンも同時に流れが悪い
・排水口から下水のような強い悪臭が上がってくる
・浴槽の排水栓を操作しても水が抜けない
・トラップ内部に異物（ピアス・指輪など）が落ちている
・床や壁にしみ・水濡れの跡が出てきた

特に、複数箇所同時の排水不良や、床や壁に染みが出ているケースは、排水管そのものの詰まり・破損・勾配不良の可能性があります。放置すると下階漏水や床下浸水につながるため、できるだけ早く点検を受けてください。

| 自分で対応しやすいケース | 業者へ相談したいケース |
| --- | --- |
| ヘアキャッチャーの髪の毛と汚れで詰まっている | 掃除・パイプクリーナーで改善しない |
| トラップ内のぬめりや軽い汚れ | 家の中の複数箇所で同時に流れが悪い |
| 重曹・クエン酸やパイプクリーナーで流れが戻る | 強い悪臭が上がり続ける |
| 浴槽排水栓のパッキンを清掃して密閉が戻る | 異物が落ちて取り出せない |
| | 床や壁に水濡れ・染みが出てきた |

お風呂の排水が悪い状態を放っておくと、悪臭・カビ・床材の劣化につながります。自分で確認してみたけれど改善しない、原因がわからず不安というときは、市販品を連用せず一度ご相談ください。

排水口の高圧洗浄から配管調査、状況によってはトラップ・配管の交換まで、年中無休・出張対応でサポートしています。
