# トイレが詰まった！原因別の対処法と業者を呼ぶべきケース

トイレの水を流したあと、なかなか水が引かない。何度レバーを回しても水位が下がらず、逆に上がってくる。そんな状況に焦った経験はありませんか。

トイレのつまりは意外とよくあるトラブルで、トイレットペーパーの使いすぎや異物の誤流入が原因になっていることも少なくありません。一方で、排水管の汚れや設備の劣化が関係しているケースもあります。

この記事では、トイレが詰まったときに考えられる原因や、自分で試せる対処法、業者へ相談したほうがよいケースについてわかりやすく解説します。

## 1. 落ち着いて状況を確認

水が引かないときは、まず落ち着いて以下のことを確認しましょう。

・水があふれそうになっていないか
・流したものに心当たりがあるか（トイレットペーパーの量、異物など）
・家中の他の排水箇所（キッチンやお風呂）も同じように流れが悪いか

他の排水箇所も詰まっている場合は、建物全体の排水管に問題がある可能性もあります。トイレ単独のつまりか、家全体の排水トラブルかを見極めることが大切です。

また、水位が便器の縁ギリギリまで上がっている場合は、あふれる前にタンクの止水バルブを閉めるか、給水元栓を止めて対応しましょう。

## 2. トイレつまりの主な原因

### 2.1. トイレットペーパーの使いすぎ

トイレットペーパーを一度に大量に流すと、水に溶けきれずに配管の中で詰まることがあります。特に節水型のトイレは流す水の量が少ないため、紙の量に敏感です。

家族が多い家庭や、来客が続いたあとはつまりやすくなるため注意が必要です。

### 2.2. 流せないものを流してしまった

トイレに流せるのは基本的にトイレットペーパーと人間の排泄物だけです。次のようなものを誤って流すと、つまりの原因になります。

・ウェットティッシュやおしりふき
・生理用品や紙おむつ
・タバコの吸い殻
・綿棒や爪楊枝
・猫砂やペットのシーツ
・大量のティッシュペーパー

「水に溶ける」と書かれていても、トイレットペーパー以外は配管の中でつまりやすいため、できるだけ流さない習慣をつけましょう。

### 2.3. 排水管の汚れや油の蓄積

長年使用しているトイレでは、排水管の内側に汚れや石鹸カス、油分が少しずつ蓄積していきます。これらが排水の流れを悪くし、ある日突然つまりとして現れることがあります。

特に使用頻度の高いトイレや、築年数の経った建物では注意が必要です。

### 2.4. 経年劣化や設備の不具合

便器や排水管自体にひび割れやズレが生じると、スムーズに流れずつまりやすくなります。また、タンク内部の部品の不具合で流す力が弱くなっているケースもあります。

これらは自分で確認するのが難しいため、気になる場合は専門業者に相談するとよいでしょう。

## 3. 試してみたい基本の対処法

安全に確認できる範囲で、次のようなことを試してみましょう。

### 3.1. ラバーカップ（すっぽん）を使う

トイレのつまりで最も一般的な対処法がラバーカップです。便器の排水口にラバーカップを密着させ、ゆっくりと押し引きを繰り返します。急に強く押すと水があふれる恐れがあるため、最初は優しく動かしてみましょう。

ラバーカップは100円ショップやホームセンターで購入できます。トイレ用のものを用意しておくと安心です。

### 3.2. お湯を流す

ぬるま湯（40度程度）をバケツに用意し、便器にゆっくりと流し入れる方法もあります。熱すぎるお湯は便器を傷める可能性があるため、必ずぬるま湯を使いましょう。

お湯を流したあと、10分から15分ほど待ってから流してみると、つまりが解消することがあります。

### 3.3. 重曹と酢を使う

重曹と酢を使った方法も、軽いつまりに効果がある場合があります。重曹を便器に振り入れ、その上から酢を注ぎます。発泡作用で排水管の汚れを浮かせてから、ぬるま湯を流すとつまりが改善することがあります。

## 4. こんなときは早めの相談を

上記の対処法を試しても改善しない場合や、次のような状況では、無理に自分で対応せず専門業者へ相談しましょう。

| 自分で確認しやすいケース | 業者へ相談したいケース |
| --- | --- |
| トイレットペーパーの使いすぎが原因と思われる | 何度試しても水が引かない |
| ラバーカップで簡単に解消した | 水があふれそう、またはあふれた |
| 重曹と酢で流れが改善した | 異物（スマホ、おもちゃなど）を落としてしまった |
| つまりが一度だけで再発しない | 何も流していないのにつまる |
| | 家中の排水が共通して流れにくい |
| | つまりを繰り返す |

迷ったときは、「安全に確認できる範囲だけ試す」のが基本です。少しでも不安がある場合は、無理をせず相談したほうが安心です。

また、以下のような対応はかえって状況を悪化させることがあるため避けましょう。

・市販の薬品（パイプ洗浄剤など）を無理に使う
・針金ハンガーなどを排水口に突っ込む
・何度も無理に流そうとレバーを操作する

薬品は種類によって便器や配管を傷めることがあり、針金などの道具は排水管を傷つける原因になります。
