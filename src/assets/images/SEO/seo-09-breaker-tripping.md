# ブレーカーが落ちる原因は？種類別の理由と再発防止のポイント

電子レンジとドライヤーを同時に使った瞬間、家中の電気が一気に消えた。あるいは、特に何もしていないのに頻繁にブレーカーが落ちて困っている。そんな経験はありませんか。

ブレーカーが落ちる原因は、単純な「電気の使いすぎ」だけではありません。分電盤にはいくつかの種類のブレーカーが並んでおり、どれが落ちているかによって、原因も対処法も変わってきます。漏電や配線の劣化が関わるケースでは、放置すると火災や感電の危険もあるため、見極めが重要です。

この記事では、ブレーカーの種類別に落ちる原因を整理し、再発を防ぐためのポイント、業者へ相談したほうがよいケースまでをわかりやすく解説します。

## 1. 分電盤にあるブレーカーの種類

分電盤の蓋を開けると、複数のスイッチが並んでいます。多くの家庭では、左から順に次の3種類が配置されています。

・**アンペアブレーカー**　➞ 契約電力（30A・40Aなど）を超えると家全体の電気を遮断する
・**漏電ブレーカー**　➞ 配線や家電から電気が漏れている時に作動する安全装置
・**安全ブレーカー（配線用遮断器）**　➞ 部屋や回路ごとに分かれており、その回路の電気使用量が上限を超えると落ちる

どのブレーカーが落ちているかで原因の見当がつきます。家全体の電気が消えていれば、アンペアブレーカーまたは漏電ブレーカーのどちらか。特定の部屋や家電だけ使えないなら、安全ブレーカーが落ちている可能性が高いです。

## 2. 種類別に考えられる落ちる原因

### 2.1. アンペアブレーカーが落ちる場合

契約アンペア数を超えて電気を使ったときに作動します。

・電子レンジ・ドライヤー・IH調理器などの大容量家電を同時に使った
・冬場の暖房ピーク時間帯（朝・夕方）に電力需要が集中している
・契約アンペア数が家族構成・家電の数に合っていない

### 2.2. 漏電ブレーカーが落ちる場合

配線や家電のどこかから電気が漏れている時に作動します。これは安全装置なので、繰り返し作動するなら必ず原因究明が必要です。

・古い家電の絶縁劣化（特に水回りで使う洗濯機・電子レンジなど）
・コンセントや配線の中に水が入り込んだ
・配線の被覆が経年劣化や小動物のかじり跡で傷ついている
・湿気の多い場所のコンセントで結露が発生している
・落雷で電気設備にダメージがあった

### 2.3. 安全ブレーカーが落ちる場合

その回路だけ電気の使いすぎ、または家電側の異常で作動します。

・1つの回路に大容量家電を集中して接続している
・タコ足配線で容量を超えている
・家電本体が故障してショートしている
・古いコンセントや配線が抵抗を持ち、過熱している

## 3. 自分でできる確認と再発防止のポイント

落ちたブレーカーの種類が分かったら、利用者側で安全に確認できることがあります。

### 3.1. 落ちたブレーカーを特定する

分電盤の蓋を開け、どのスイッチが下がっているかを目視で確認します。漏電ブレーカーは「漏電」表示が出ているタイプが多く、見分けやすくなっています。

### 3.2. 使っていた家電を整理する

アンペアブレーカーや安全ブレーカーが落ちた場合は、その直前に使っていた家電を思い出します。大容量家電を同じ時間帯・同じ回路で重ねて使っていなかったかを振り返り、使うタイミングを分けるだけで再発を防げることがあります。

### 3.3. 安全に復旧する手順

アンペアブレーカー・安全ブレーカーの場合は、原因の家電のスイッチを切ってからブレーカーを上げ直します。

漏電ブレーカーが落ちた場合は手順が異なり、自己判断で何度も上げ下げしてはいけません。詳細な復旧手順や、漏電箇所の特定方法は別記事「ブレーカーが上がらない時の原因と安全に復旧する手順」で解説していますので、合わせて参照してください。

### 3.4. 再発を防ぐ日常のポイント

ブレーカーが落ちにくくする工夫として、次のような対策があります。

・大容量家電を同時に使わないよう、家族で時間帯を調整する
・タコ足配線を避け、壁のコンセントから直接使う
・冬場・夏場のピーク時間は、消費電力の大きい家電の使用を分散する
・契約アンペア数が生活実態に合っているか見直す
・古い家電は、絶縁劣化の兆候（焦げ臭さ・触ると熱い）がないか定期的に確認する

前述のように漏電ブレーカーが繰り返し落ちる場合は、家電や配線のどこかに異常がある可能性が高いため、利用者側での対応では追いつきません。安全のため、専門業者に点検を依頼するのが現実的です。

## 4. こんなときは早めの相談を

次のような状況に当てはまる場合は、自分で繰り返し復旧させようとせず、早めに業者へ相談することをおすすめします。

・漏電ブレーカーが繰り返し作動する
・特定の家電を使うとすぐにブレーカーが落ちる
・コンセントや家電本体が焦げ臭い、触ると熱い
・分電盤から焦げたようなにおいがする
・水濡れや湿気のあとからブレーカーが落ちるようになった
・落雷や停電のあとに不調が続いている
・分電盤が古く、ブレーカーの操作感が固い・戻らない

特に、焦げ臭さや煙、火花を伴う場合は、火災の危険があります。すぐに分電盤の主幹ブレーカーを切り、使っていた家電のプラグを抜いた状態で業者を呼んでください。漏電は目に見えないため、放置すると感電や火災に直結する可能性があります。

| 自分で対応しやすいケース | 業者へ相談したいケース |
| --- | --- |
| 大容量家電の同時使用でアンペアブレーカーが落ちた | 漏電ブレーカーが繰り返し作動する |
| 安全ブレーカーが落ち、家電を減らせば復旧する | 特定の家電やコンセントから焦げ臭さがある |
| タコ足配線を整理して再発を防げる | 分電盤から焦げたにおいや異音がする |
| 契約アンペア数の見直しを検討する | 水濡れ・落雷の後から不調が続く |
| | 古い分電盤でブレーカーが正常に作動しない |

ブレーカーが頻繁に落ちる状態を放置すると、家電の故障だけでなく、火災や感電のリスクにもつながります。何度復旧させてもまた落ちる、原因がわからず不安というときは、無理に繰り返さず一度ご相談ください。

漏電点検から分電盤の交換・回路の見直しまで、年中無休・出張対応でサポートしています。
