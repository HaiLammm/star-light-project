「シンクの下から水が漏れている…」「見えない場所から水が染み出してきている」など、排水管からの水漏れに気づくと焦ってしまいますよね。

排水管からの水漏れは、パッキンの劣化やナットの緩みといった簡単な原因で起きることもあれば、配管自体の破損や詰まりが関係しているケースもあります。放置すると床や家具の腐食、カビの発生、さらには下の階への被害にもつながりかねません。

この記事では、排水管からの水漏れが発生したときに確認したいポイント、場所別の主な原因、そして緊急時に自分でできる応急処置について詳しく解説します。


## 排水管の水漏れを見つけたらまず確認すること

排水管からの水漏れに気づいたら、慌てずにまずは被害の拡大を防ぎ、状況を把握することが大切です。

### 水漏れの場所と範囲を特定する

水漏れが発生している場所を特定することは、原因を突き止める第一歩です。どこから、どのくらいの量の水が漏れているかを確認しましょう。

- **水漏れ箇所**：蛇口の根元、シンク下、洗面台下、洗濯機の下、お風呂の排水溝周辺など。
- **水漏れの症状**：水がポタポタと垂れているのか、じわじわと染み出しているのか、勢いよく噴き出しているのか。
- **濡れている範囲**：水が床に広がっているか、壁が濡れているか。
- **水の使われ方と連動しているか**：水を流しているときだけ漏れるのか、常に漏れているのか。

水漏れ箇所が分かりにくい場合は、キッチンペーパーやティッシュで排水管の周囲を拭いてみると、どこから水が出ているか特定しやすくなります。

### 水を止める

水漏れ箇所を特定したら、すぐに水を止めましょう。水の供給を遮断することで、それ以上の被害拡大を防ぐことができます。

- **特定の蛇口からの水漏れの場合**：その蛇口の**止水栓**を閉めます。止水栓はシンク下や洗面台下に設置されていることが多いです。
- **家全体で水漏れしている場合や止水栓が見つからない場合**：水道メーターの近くにある**元栓**を閉めます。元栓を閉めると家中の水が止まるため、注意してください。
- **給湯器関連の水漏れの場合**：給湯器の止水栓も閉めるか、給湯器の電源を切ります。

水を止める際は、無理な力を加えず、ゆっくりと閉めましょう。固着している場合は無理に回さず、次の応急処置に進んでください。

### 応急処置を施す

水を止めたら、次に水漏れによる被害を最小限に抑えるための応急処置を行いましょう。

- **漏れている水をバケツやタオルで受ける**：床が濡れるのを防ぎ、被害を軽減します。
- **排水管の破損箇所に防水テープやビニール袋を巻く**：一時的に水漏れを抑えることができます。これはあくまで一時的な処置であり、根本的な解決にはなりません。
- **濡れた場所を拭き取る**：カビの発生や建材の劣化を防ぐため、可能な範囲で水気を拭き取ります。

応急処置を施したら、早めに専門業者に相談し、適切な修理を依頼しましょう。

## 排水管から水漏れする主な原因

排水管からの水漏れは、設置場所や状況によってさまざまな原因が考えられます。ここでは、よくある原因を紹介します。

### 接続部分の緩みやパッキンの劣化

排水管の水漏れで最も多い原因の一つが、接続部分の緩みやパッキンの劣化です。

- **ナットの緩み**：排水管と排水口、または排水管同士の接続部分にあるナットが緩んでいる。
- **パッキンの劣化**：接続部分の隙間を埋めるゴム製パッキンが、経年劣化で硬化したりひび割れたりしている。

これらの原因は、長年の使用や振動、温度変化によって生じることが多く、比較的簡単に修理できるケースもあります。

### 排水管の詰まりによる逆流・溢れ

排水管内部に油汚れや髪の毛、石鹸カスなどが溜まって詰まると、水の流れが悪くなり、逆流や溢れを起こして水漏れにつながることがあります。

- **シンク下のS字トラップ内の詰まり**：食べカスや油汚れが蓄積している。
- **洗濯機の排水ホースと排水口の接続部分の詰まり**：糸くずや髪の毛が絡まっている。
- **排水管自体の奥深くでの詰まり**：長年の使用による固形物の堆積。

詰まりが原因の場合、水を使用するたびに水が溢れたり、流れが悪くなったりといった症状が現れます。

### 配管自体の破損や腐食

長年使用している排水管は、素材の劣化や外部からの衝撃、凍結などによって破損したり腐食したりすることがあります。

- **経年劣化によるひび割れや穴あき**：特に古い金属製配管に多い。
- **外部からの衝撃**：物をぶつけたり、配管に負荷がかかったりした場合。
- **薬品による腐食**：強力な排水管洗浄剤の使いすぎや誤用。

配管自体の破損や腐食は、広範囲に水漏れを引き起こす可能性があり、専門的な修理が必要です。

### 排水トラップの劣化・破損

排水トラップは、下水からの悪臭や害虫の侵入を防ぐための重要な部品です。このトラップ自体が劣化したり破損したりすると、水漏れの原因となります。

- **トラップ本体のひび割れ**：特にプラスチック製のトラップで、経年劣化により硬化し破損することがある。
- **トラップ内部の部品の劣化**：封水カップの劣化など。

排水トラップからの水漏れは、目に見えやすい場所で発生することが多いため、比較的早期に発見しやすいです。

### 凍結による配管の破裂（冬場）

寒冷地や冬場の冷え込みが厳しい地域では、排水管内の水が凍結し、体積が膨張することで配管が破裂することがあります。

- **露出している配管や屋外の配管**：特に凍結しやすい。
- **長期間留守にする際の対策不足**：水抜きを怠った場合。

凍結による破裂は、突然大量の水漏れを引き起こすことがあり、被害が大きくなる傾向があります。

## 場所別の確認ポイントと応急処置

水漏れの場所によって、確認すべきポイントや応急処置の方法が異なります。

### シンク下（キッチン）の排水管からの水漏れ

キッチンシンク下の排水管は、食べカスや油汚れが原因で詰まりやすく、水漏れも頻繁に起こります。

- **確認ポイント**：
  - シンク下の収納を開け、排水パイプや接続部分に水滴がないか、濡れた跡がないかを確認。
  - S字トラップの接続ナットが緩んでいないか、パッキンが劣化していないかを目視でチェック。
  - 水を流したときに、どこから水が漏れてくるかを特定。
- **応急処置**：
  - 緩んでいるナットがあれば、ゆっくりと締めてみる。強く締めすぎると破損の原因になるので注意。
  - パッキンの劣化が明らかな場合は、ホームセンターで購入し交換を試みる。サイズを間違えないように古いパッキンを持参する。
  - 排水管のひび割れなどには、防水テープをしっかりと巻き付ける。

### 洗面台下の排水管からの水漏れ

洗面台下も、髪の毛や石鹸カス、歯磨き粉などが詰まりやすく、接続部分からの水漏れが起こりがちです。

- **確認ポイント**：
  - 洗面台下の扉を開け、排水パイプ全体を目視で確認。特に壁への接続部分やトラップ周りに注意。
  - 排水栓（ポップアップ式など）の根元や、オーバーフロー（溢れ止め）からの水漏れがないかも確認。
- **応急処置**：
  - キッチンと同様に、緩んでいるナットを締める、防水テープを巻くといった処置を行う。
  - 洗面ボウルのオーバーフローから水漏れがある場合は、排水栓のゴムパッキンの劣化が考えられるため、交換を検討する。

### お風呂場の排水管からの水漏れ

お風呂場は、床下など見えにくい場所での水漏れが多く、発見が遅れると被害が大きくなることがあります。

- **確認ポイント**：
  - 浴槽の排水口の蓋を開け、ヘアキャッチャー（ゴミ受け）やトラップ部分に髪の毛や石鹸カスが詰まっていないか確認。
  - 浴槽の側面やエプロン内部（取り外せる場合）に水が染み出していないか確認。
  - シャワーを使用しているときに、壁や床との隙間から水が漏れていないか。
- **応急処置**：
  - ヘアキャッチャーやトラップの掃除で詰まりが解消されることも多い。
  - 見えにくい場所からの水漏れは応急処置が難しいため、すぐに専門業者に連絡する。

### 洗濯機の排水管からの水漏れ

洗濯機の排水管は、洗濯時の振動や排水量の多さから、ホースの接続部分が外れたり、詰まりが原因で溢れたりすることがあります。

- **確認ポイント**：
  - 洗濯機本体を動かし、排水ホースと排水口（または排水トラップ）の接続部分がしっかり締まっているか確認。
  - 排水ホース自体に穴が開いていないか、劣化していないかを目視でチェック。
  - 排水口の周りに洗濯槽から出た糸くずや髪の毛が詰まっていないか。
- **応急処置**：
  - 排水ホースが外れている場合は、しっかりと奥まで差し込み、ホースバンドで固定する。
  - ホースに穴が開いている場合は、防水テープで補修するか、新しいホースに交換する。
  - 排水口の詰まりは、重曹とクエン酸、または市販のパイプクリーナーで解消できる場合がある。

## こんなときは早めの相談を

排水管の水漏れは、自分でできる応急処置もありますが、根本的な解決には専門的な知識や技術、工具が必要なケースがほとんどです。特に以下のような状況が見られる場合は、無理な自己判断や修理は避け、早めに専門業者へ相談しましょう。

| 自分で対処しやすいケース | 業者へ相談したいケース |
| :----------------------- | :----------------------- |
| 接続ナットの緩み         | 応急処置しても漏れが止まらない |
| パッキンの軽度な劣化     | 配管自体にひび割れや破損がある |
| 排水口の軽度な詰まり     | 床下や壁の内部から水漏れがある |
| ホースバンドの外れ       | 下の階へ水漏れ被害が出た |
| 洗濯機の排水ホース破損   | 原因が特定できない、見えない場所からの漏れ |
| ―                        | 強力なパイプクリーナーを誤用した |

特に、水漏れ箇所が特定できない、壁や床の内部から水が染み出している、下の階へ水漏れ被害が出た、といった場合は、状況が深刻化している可能性があります。また、配管自体の交換や修理には、専門的な資格が必要な作業も含まれます。無理に自分で対応しようとすると、かえって被害を拡大させたり、新たなトラブルを引き起こしたりするリスクがあります。
