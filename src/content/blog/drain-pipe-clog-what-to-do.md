「シンクの水がなかなか流れない…」「お風呂の排水口からゴボゴボと音がする」といった排水管の詰まりは、毎日の生活に大きなストレスを与えます。放置すると異臭や逆流、さらには水漏れといった深刻なトラブルに発展する可能性もあります。

排水管の詰まりは、髪の毛や油汚れ、石鹸カスなど、日々の生活で排出されるものが蓄積することで発生します。詰まりの原因や場所によっては自分で対処できるケースも少なくありませんが、無理な作業はかえって状況を悪化させることもあるため注意が必要です。

この記事では、排水管が詰まったときにまず確認したいポイントと、考えられる主な原因、自分でできる安全な対処法、そして専門業者に相談すべきケースについて詳しく解説します。

## 排水管が詰まったときに順番に確認したいこと

排水管の詰まりは、焦って無理な処置をすると状況を悪化させることがあります。まずは落ち着いて、詰まりの原因と状況を正確に把握するためのチェックを行いましょう。



![ao-dai-gam-hoa](/images/SEO/drain-pipe-clog-what-to-do/1.png)


### 症状の範囲と発生箇所を特定する

排水の流れが悪いとき、まず確認したいのは「家全体で水が流れないのか」「特定の場所だけなのか」という点です。これにより、詰まりの場所が排水口付近なのか、それとももっと奥の共有排水管なのかを判断できます。

- **家中の排水が悪い場合**
  ➞ メインの排水管や下水本管、または屋外排水設備のトラブルが考えられます。複数の排水口で同時に症状が出ている場合は、より専門的な対応が必要になる可能性が高いです。
- **特定の排水口だけが詰まっている場合**
  ➞ その排水口から奥の配管にかけての詰まりが考えられます。キッチンなら油汚れ、お風呂なら髪の毛や石鹸カスが原因であることが多いです。

どこで詰まっているかを特定できれば、より適切な対処法を選ぶことができます。普段から流れが悪い場所がないか、日頃から意識しておくのも良いでしょう。

### 排水口やワントラップの清掃状況を確認する

排水口のすぐ下には、悪臭や害虫の侵入を防ぐための「ワントラップ」と呼ばれる部品があります。このワントラップやその周辺にゴミが溜まっていると、水の流れが悪くなります。

- **排水口のフタやゴミ受けを外し、目に見えるゴミや髪の毛を取り除く**
  ➞ 取り除く際は、使い捨ての手袋などを使用しましょう。
- **ワントラップを外し、裏側や周辺のヌメり、ヘドロ、石鹸カスなどを確認する**
  ➞ 長年掃除していない場合、これらがびっしり付着していることがあります。
- **歯ブラシや使い古しのスポンジなどで、きれいに洗い流す**
  ➞ 熱湯をかけると油汚れが溶けることがありますが、PVC製の排水管が変形する可能性もあるため、**必ずぬるま湯を使用してください**。

目に見える部分の汚れを取り除くだけで、詰まりが解消することもあります。この作業は定期的に行うことで、深刻な詰まりを予防できます。

### 排水管の詰まりの程度を確認する

水の流れが悪いといっても、まったく流れないのか、ゆっくりと流れるのかで詰まりの深刻度が異なります。これにより、自分で対処できる範囲かどうかの目安になります。

- **水が少しずつでも流れる場合**
  ➞ 軽度の詰まりである可能性が高く、自分でできる対処法で改善する可能性があります。ただし、流れが悪いまま放置すると悪化するリスクもあります。
- **水がまったく流れず、逆流してくる場合**
  ➞ 重度の詰まりである可能性が高いです。無理に自分で対処しようとせず、早めに専門業者に相談することを検討しましょう。

水を流してみて、ゴボゴボといった異音や、普段と異なる泡の立ち方がないかも確認しておくと良いでしょう。これらの症状は、空気の抜けが悪くなっているサインかもしれません。

## それでも改善しないときに考えられる主な原因

排水口やワントラップの清掃、詰まりの程度の確認をしても水が流れない場合、より奥の排水管に問題があると考えられます。ここでは、自分で対処が難しい主な原因を解説します。

### 長期間蓄積された油汚れや石鹸カス

キッチンの排水管で特に多いのが、油汚れの蓄積による詰まりです。油は冷えると固まり、排水管の内側に付着して水の通り道を狭めます。また、お風呂や洗面所の排水管では、石鹸カスや皮脂汚れが主な原因となります。

- **キッチンのシンクで水が流れにくい場合**
  ➞ ラーメンの残り汁や揚げ物の油をそのまま流すことが多かった、という経験があれば、油汚れが原因の可能性が高いです。これらの油が冷えて固まり、他のゴミを絡め取って塊となります。
- **お風呂や洗面所で流れが悪い場合**
  ➞ 石鹸、シャンプー、リンスのカスなどが排水管の内側に付着し、そこに髪の毛が絡みつくことでヘドロ状の詰まりを形成します。これは特に見えにくい場所でゆっくりと進行するため、気づきにくいことがあります。

これらの汚れは、市販のパイプクリーナーでは完全に除去しきれないほど強固に固まっている場合があり、専門的な高圧洗浄などが必要になることがあります。

### 髪の毛や異物の絡まり

お風呂や洗面所の排水管詰まりの最大の原因の一つは、やはり髪の毛です。髪の毛は水に溶けないため、ワントラップや排水口の奥のS字トラップなどに絡まりつき、他のゴミや石鹸カスを巻き込んで巨大な塊となることがあります。

- **シャワーを浴びている際に、すぐに足元に水が溜まる**
  ➞ 排水口から少し奥の場所に髪の毛が密集して詰まっている可能性が高いです。
- **洗面台でヒゲを剃ったり、ペットを洗ったりすることが多い**
  ➞ 細かい毛が流れ込み、排水管のカーブ部分で詰まりやすい傾向があります。
- **小さな固形物を誤って流してしまった**
  ➞ アクセサリー、子どものおもちゃ、歯磨き粉のキャップなど、固形物が排水管の途中で引っかかり、その周りに髪の毛や汚れが絡みついて詰まりを引き起こすことがあります。

固形物が原因の場合、自分で無理に取ろうとすると、さらに奥へ押し込んでしまったり、配管を傷つけたりするリスクがあるため、慎重な対応が必要です。

### 配管自体の老朽化や破損

築年数が古い建物では、排水管そのものが老朽化している可能性も考えられます。配管内部の劣化により、表面がザラザラになったり、ひび割れが生じたりすることで、汚れが付着しやすくなり、詰まりの原因となることがあります。

- **長年の使用による配管のサビや腐食**
  ➞ 特に金属製の配管の場合、サビが発生し、内径が狭くなることで水の流れが悪くなります。サビの一部が剥がれて詰まりの原因になることもあります。
- **地震などの外的要因による配管のズレや破損**
  ➞ わずかなズレでも段差が生じ、ゴミが引っかかりやすくなります。破損している場合は、水漏れにつながる非常に深刻な問題です。
- **排水管の勾配不良や設計ミス**
  ➞ 稀なケースですが、新築時やリフォーム時に排水管の勾配が適切でないと、常に水が滞留しやすく、詰まりが発生しやすい状態になります。

配管自体の問題は、自分で確認したり修理したりすることはほぼ不可能です。専門業者による調査と修理、場合によっては配管の交換が必要になります。賃貸物件の場合は、管理会社や大家さんに報告しましょう。

## 自分でできる安全な対処法

排水口の詰まりは、状況によっては自分で解消できる場合があります。無理な力を加えたり、危険な薬剤を不適切に使用したりしないよう、安全に注意しながら試してみましょう。

### 重曹とクエン酸（またはお酢）を使う

軽度の詰まりや排水管のヌメり解消には、重曹とクエン酸（またはお酢）を組み合わせた方法が効果的です。化学反応で発生する泡が汚れを浮かせてくれます。

1. **排水口のゴミや髪の毛を取り除く**
   ➞ 清掃状況を確認し、目に見える汚れは先に取り除いておきましょう。
2. **重曹をカップ1/2〜1杯程度、排水口に直接振り入れる**
   ➞ 排水管の壁面に付着するよう、広めに振り入れるのがポイントです。
3. **クエン酸をカップ1/2程度、またはお酢を同量程度、重曹の上からゆっくりと流し込む**
   ➞ 重曹とクエン酸が反応し、泡が発生し始めます。
4. **30分〜1時間ほど放置する**
   ➞ この間は水を流さず、反応を待ちます。
5. **ぬるま湯（50℃程度）をゆっくりと流し、洗い流す**
   ➞ 熱湯は排水管を傷める可能性があるため、**必ずぬるま湯を使用してください**。

この方法は、特にキッチンの油汚れやお風呂の石鹸カス、ヌメりに有効です。定期的に行うことで、詰まりの予防にもなります。

### パイプクリーナーを使用する

市販のパイプクリーナーは、髪の毛や油汚れを溶かす効果があるため、自分でできる対処法として手軽です。製品によって成分や使用方法が異なるため、**必ず製品の指示に従ってください**。

1. **排水口のゴミや髪の毛を取り除く**
   ➞ パイプクリーナーの効果を最大限に引き出すため、前準備が重要です。
2. **製品の指示に従って、適量のパイプクリーナーを流し込む**
   ➞ 一度に大量に投入しても効果は変わりません。
3. **規定の時間（30分〜数時間）放置する**
   ➞ 詰まりがひどい場合は、長時間放置すると効果が高まることがあります。ただし、必要以上に長時間放置すると、排水管を傷める可能性もあるため注意が必要です。
4. **大量の水を勢いよく流し、完全に洗い流す**
   ➞ 洗い残しがあると、薬剤が残って配管を傷めたり、他の薬剤と混ざって有害ガスが発生したりする原因になります。

**【注意点】**
- **換気をしながら作業を行う**：薬剤の蒸気を吸い込むと危険です。
- **ゴム手袋、保護メガネを着用する**：皮膚や目に入ると炎症を起こす可能性があります。
- **他の薬剤と混ぜない**：特に酸性の洗剤と混ぜると、有毒ガスが発生する危険があります。
- **熱湯を流さない**：薬剤と反応して有害ガスが発生したり、排水管が変形したりする可能性があります。

### ラバーカップ（すっぽん）を使う

水がまったく流れない、または溜まってしまうような重度の詰まりには、ラバーカップ（いわゆる「すっぽん」）が有効です。水圧を利用して詰まりを吸い出す、または押し流す効果があります。

1. **排水口の周りに水を溜める**
   ➞ ラバーカップのゴム部分が完全に水に浸かるように、水を溜めてください。水が少ない場合は、水を足しましょう。
2. **ラバーカップを排水口に密着させる**
   ➞ 空気が漏れないように、しっかりと密着させることがポイントです。
3. **ゆっくりと押しつけ、勢いよく引き上げる動作を数回繰り返す**
   ➞ 「引く」ときに詰まりが動きやすいと言われています。
4. **水が流れるようになったら、しばらく水を流し続けて完全に洗い流す**

**【注意点】**
- **トイレ用と排水口用を使い分ける**：形状が異なるため、適切なものを使用しましょう。
- **無理な力を加えない**：配管を破損させる可能性があります。
- **異物が詰まっている場合は無理に使用しない**：さらに奥へ押し込んでしまうことがあります。

### ワイヤーブラシを使う

排水口から少し奥の場所、特にS字トラップなどで詰まっている場合は、ワイヤーブラシ（パイプクリーナーブラシ）が効果的なことがあります。

1. **排水口のゴミ受けやワントラップを取り外す**
   ➞ ワイヤーブラシがスムーズに入るようにします。
2. **ワイヤーブラシをゆっくりと排水管に入れていく**
   ➞ 詰まっている場所に到達したら、少し抵抗を感じるはずです。
3. **ワイヤーブラシを回転させたり、前後に動かしたりして、詰まりをかき出す**
   ➞ 無理に奥まで押し込まず、詰まりが解除される感触があるまでゆっくりと行いましょう。
4. **詰まりが取れたら、水を流して確認する**

**【注意点】**
- **無理に押し込まない**：配管を傷つけたり、ワイヤーブラシが途中で引っかかって取れなくなったりする可能性があります。
- **使用後は必ずブラシをきれいに洗浄する**：衛生的に保ち、次回も使えるようにしましょう。
- **異物による詰まりには効果が薄い場合がある**：特に固形物の場合、ワイヤーブラシでは対応できないことがあります。

## こんなときは早めの相談を

上記の対処法を試しても排水管の詰まりが改善しない場合や、次のような症状が見られる場合は、無理に自分で対処しようとせず、早めに専門業者へ相談しましょう。放置すると、より大きなトラブルにつながる可能性があります。

| 自分で対処しやすいケース | 業者へ相談したほうがよいケース |
| --- | --- |
| 軽度のヌメりや髪の毛詰まり | 全く水が流れない・逆流する |
| 重曹やパイプクリーナーで改善 | 複数の排水口で同時に詰まり |
| ラバーカップで一時的に改善 | 異臭がひどい、または継続する |
| 定期的な清掃での予防 | 配管から異音や振動がする |
| ワントラップやゴミ受けの汚れ | 異物を流してしまった可能性がある |
| ― | 排水管の周りで水漏れしている |
| ― | 築年数が古い建物での詰まり |

特に、**排水管の周りから水漏れしている場合**や、**異臭がひどく健康被害が心配される場合**、**異物を流してしまい詰まっている可能性がある場合**は、速やかに専門業者に連絡することが重要です。自己判断で無理に作業を続けると、配管を破損させたり、状況を悪化させたりするリスクが高まります。
