ブレーカーが突然落ち、あわてて復旧しようとしたけれどスイッチが上がらない。上げてもすぐに「バチン」と落ちてしまう。誰しも一度は経験したことがあるかもしれません。特に夜間や悪天候のときは、不安で頭が真っ白になってしまいがちです。

まず、もしブレーカーが落ちた直後で、まだ基本的な復旧手順を試していないなら、先に「[ブレーカーが落ちたらまず試す確認手順と安全な復旧方法](/columns/breaker-trip-first-steps/)」をお読みください。この記事では、その基本的な手順を一通り試してもなお**復旧できない**、または**すぐにまた落ちてしまう**ケースに焦点を当て、より踏み込んだ安全確認と、原因を絞り込む方法を解説します。

## ブレーカーが戻らないときにまず行う安全確認

すでに基本的な復旧を試してもダメだった場合、ここからの手順は焦らず、一つずつ順に行ってください。感電や火災を防ぐため、**濡れた手で触らない、床が濡れている場所で操作しない**ようにしてください。

### 落ちているブレーカーの種類を確認する

分電盤を開け、どの種類のブレーカーが落ちているか、または上がらなくなっているかを観察します。ブレーカーには主に以下の３種類があり、落ちている種類によって原因と対処が大きく変わります。

・**アンペアブレーカー（サービスブレーカー）**  
契約アンペアを超えた場合に落ちる、左端などにある大きなスイッチ。同時に複数の電気機器を使いすぎたときに動作します。

単純な使いすぎなら、いくつか機器を抜けば復旧するはずです。それでも戻らない場合は、内部の故障や漏電による二次的なトリップが考えられます。

・**漏電遮断器（漏電ブレーカー）**  
赤や黄色のテストボタンがついており、漏電を感知して回路を遮断します。一度復旧できても再度落ちる場合、家のどこかで漏電が起きている可能性が高いです。

・**安全ブレーカー（配線用遮断器）**  
各部屋や回路ごとに分かれた小さなスイッチ。短絡や過負荷で落ちます。このうち一つだけが上がらない場合、その回路に問題があります。

### すべての電気機器を抜き、安全ブレーカーをすべてOFFにする

原因を確実に切り分けるために、家中のコンセントからすべての電気機器のプラグを抜きます。スイッチが切れているだけでは不十分で、**必ずプラグを物理的に抜くことです**。スイッチが切れていても内部で微弱な漏電が起こることがあるため、完全に切り離す必要があります。

・すべての部屋のコンセントからプラグを抜く  
・エアコンや冷蔵庫など、普段は抜かない機器もしばらくの間だけ抜く（冷蔵庫内の食品はクーラーボックスなどで保管してください）  
・分電盤の、落ちているブレーカー以外の安全ブレーカーも含めて、すべてOFFにする

ここで、漏電遮断器やアンペアブレーカーはそのまま（ONまたはOFFの状態で）にし、まず負荷側を完全にゼロにすることが目的です。

### ブレーカーを順に復旧させて問題の回路を見つける

負荷をすべて抜き、安全ブレーカーをすべてOFFにしたら、次の手順で復旧を試みます。

・**最初にアンペアブレーカー（または漏電遮断器）をONにする**  
　これがONにならなければ、ブレーカー自体の故障か、ブレーカーより前（電力会社側）のトラブルが疑われます。  
・次に、安全ブレーカーを一つだけONにし、問題なければ次の安全ブレーカーをONにする、というふうに順に復旧させていく  
・ある安全ブレーカーをONにした瞬間に、漏電遮断器またはその安全ブレーカーが再び落ちた場合、その回路に問題があることが確定する  
・ONにしても落ちなかった回路については、少し時間を置いてから（機器を接続せずに）様子を見る

落ちた回路の名称（「台所」「2階洋室」など）をメモしておくと、後の修理依頼のときに役立ちます。どの安全ブレーカーをONにしても落ちないのに、機器をつなぐと落ちる場合は、その機器自体の故障を疑います。

## それでも復旧しないときに考えられる原因

前章のチェックで問題の回路が絞り込めなかったり、何度試してもブレーカーが上がらない場合、以下の４つの原因が考えられます。ここでは原因の理解を深めていただくために、現象とともに解説します。いずれも内部の電気的な問題ですので、**分解や修理は絶対に避けてください**。

### 短絡（ショート）の深刻な状態

安全ブレーカーをONにした瞬間に「バチン」と大きな音がして落ちる、またはスイッチを上げようとしても手応えがなく跳ね返される現象が見られます。火花が散った痕跡（黒いスス）がコンセントやプラグに見られることもあります。  
これは回路内の電線同士が直接接触して大電流が流れている「短絡」の状態です。コンセント内部で配線が緩んで接触した、壁の中で電線が傷ついた、工事中のドリルが配線を損傷したといったことが考えられます。
短絡が起きている回路を使い続けると、発熱から**火災**に至る危険性が非常に高いため、該当回路の安全ブレーカーはOFFのままにしてください。

### 漏電が発生している

漏電遮断器だけが落ちて安全ブレーカーは落ちていない、または雨の日や湿気の多い日だけブレーカーが落ちやすいといった現象がみられます。漏電遮断器のテストボタンを押すと正常に作動することを確認できます。  
原因は、電気が本来流れるべき道から外れて建物の壁や地面に流れ出る「漏電」です。古いエアコンや洗濯機など水まわり家電の絶縁劣化、屋外の防水コンセントへの浸水、壁内配線の劣化などが主な要因です。
漏電は感電事故や漏電火災につながるため、早急な対応が必要です。

### ブレーカー自体の故障

どの安全ブレーカーもONになっておらず、家中のプラグもすべて抜いているのに、アンペアブレーカーや漏電遮断器が上がらない現象が起きます。ブレーカー本体から「ウーン」という異音がする、触れると熱い、焦げたようなにおいがすることもあります。  
ブレーカー内部の接点の摩耗や熔着、ばねのへたり、電子部品の故障が原因です。築年数が経過した住宅では、ブレーカーが一度も交換されていないことも多く、内部劣化による動作不良が起こります。
**この状態で無理にONにすると、ブレーカー自体が発火する危険性があります。**

### 停電や外部の電源トラブル

ブレーカーがまったく操作できず（ON/OFFにかかわらず）、家中の電気がつかない。分電盤のどのスイッチも手応えが変わらず、周辺の家も停電している（夜間なら窓の外を見て確認）という場合は、外部のトラブルが疑われます。  
電力会社の送電線トラブルや、電柱の変圧器の故障、地域一帯の計画停電などが原因として考えられます。台風や大雪などの悪天候時に発生しやすいです。この場合、一般家庭の分電盤ではどうすることもできません。  
もし周りは電気がついているのに自宅だけ停電している場合は、屋内配線の根本的な問題か、引き込み線の断線の可能性があります。いずれにせよ専門業者または電力会社への連絡が必要です。

## こんなときはすぐに専門業者へ

これまでに紹介した安全確認を丁寧に行っても復旧しない、あるいは以下のような危険な兆候がある場合は、ただちに使用を中止し、専門の電気工事士を呼んでください。電気設備の不適切な取り扱いは、消防庁も注意喚起する[電気火災のリスク](https://www.fdma.go.jp/publication/hakusho/r6/chapter1/section1/para2/68016.html)を高めます。

**すぐに対処が必要な危険サイン：**
- ブレーカーから煙や火花が出ている
- 焦げたにおいや異常な熱がある
- つなぐ機器が何もないのにブレーカーが上がらない
- 何度も落ちる、または復旧後すぐにまた落ちる
- 漏電遮断器がどうしても戻らない

**絶対にやってはいけないこと：**
- **ブレーカーを何度も強引にONにしようとする**: 内部の損傷を悪化させ、発火のリスクを高めます
- **分電盤のカバーを開けて内部を掃除する、ネジを触る**: 感電や短絡の危険があります。電気工事士の資格がない限り、絶対に分解しないでください
- **濡れた手や濡れた床で操作する**: 感電事故の原因になります
- **落ちたブレーカーをテープで固定するなどの強制的な通電**: 非常に危険で、火災や法令違反につながります

以下の表を目安に、ご自身で様子を見るケースと業者を呼ぶケースを判断してください。

| 自分で対応しやすいケース | 業者へ相談したいケース |
| --- | --- |
| 特定の機器を抜いたら復旧した | 機器を全部抜いてもブレーカーが上がらない |
| 安全ブレーカー１つのみ落ち、他の回路は正常 | 漏電遮断器がどうしても戻らない |
| タコ足配線をやめたら落ちなくなった | ブレーカーから煙・におい・異音がする |
| 過負荷が原因で、使い方を改めることで再発防止できる | 築年数が経過し、一度もブレーカー交換の履歴がない |
|  | 落ちる原因がまったくわからない、不安が大きい |

ブレーカーが上がらない状態を放置すると、たとえ一時的に復旧できても、根本的な問題が解消されていなければ、突然の停電や火災のリスクが続くことになります。特に漏電や短絡が疑われるケースでは、見えない部分で進行する危険があるため、少しでも不安を感じたら専門家に点検を依頼することが、あなたと家族の安全を守る最善の方法です。
